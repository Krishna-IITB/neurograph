# Sample PDFs

Drop `.pdf` files in this directory and run `make ingest` to add them to the indexes.

For the bundled demo, we recommend Wikipedia article exports of:

- Elon Musk
- Tesla, Inc.
- SpaceX
- SolarCity

Export route on Wikipedia: **Tools → Download as PDF**.

> The `.gitignore` excludes actual PDFs from the repo. They live locally only.
